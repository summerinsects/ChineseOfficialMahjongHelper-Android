# ChineseOfficialMahjongHelper-Android 国标小助手

- 这是与国标麻将相关的应用，包含算番器、线下实麻计分器、番种详细说明、牌理等内容
- 原项目：[国标小助手](https://github.com/summerinsects/ChineseOfficialMahjongHelper)
- 算法库：https://github.com/summerinsects/mahjong-algorithm
- 由于cocos2d-x不再维护，加之iOS上架麻烦，新项目决定放弃iOS，使用Android原生重写
